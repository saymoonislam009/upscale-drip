🗄 Supabase Database Setup

Supabase is a free cloud database. Once set up, your products will show on the website from any device, anywhere. Follow these steps once:

1
Go to supabase.com → Sign up free → Create a new project (pick any name, e.g. "upscale-drip")
2
In your project, go to SQL Editor and run this SQL to create the products table:

CREATE TABLE products ( id BIGSERIAL PRIMARY KEY, name TEXT NOT NULL, cat TEXT NOT NULL, img TEXT NOT NULL, price INTEGER NOT NULL, desc TEXT, isnew BOOLEAN DEFAULT false, sizes JSONB DEFAULT '[]' ); ALTER TABLE products ENABLE ROW LEVEL SECURITY; CREATE POLICY "Public read" ON products FOR SELECT USING (true); CREATE POLICY "Service write" ON products FOR ALL USING (true);
3
